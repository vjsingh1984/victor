## üìã Next Steps to Rebrand

### Quick Rename Script:

```bash
#!/bin/bash
# rename_to_victor.sh

OLD_NAME="victor"
NEW_NAME="victor"  # or "singhai" or "vijay"

# 1. Rename directory
mv $OLD_NAME $NEW_NAME

# 2. Update pyproject.toml
sed -i '' "s/name = \"$OLD_NAME\"/name = \"$NEW_NAME\"/g" pyproject.toml

# 3. Update imports in all Python files
find . -name "*.py" -exec sed -i '' "s/from $OLD_NAME/from $NEW_NAME/g" {} +
find . -name "*.py" -exec sed -i '' "s/import $OLD_NAME/import $NEW_NAME/g" {} +

# 4. Update README
sed -i '' "s/$OLD_NAME/$NEW_NAME/g" README.md

# 5. Update all docs
find . -name "*.md" -exec sed -i '' "s/$OLD_NAME/$NEW_NAME/g" {} +

# 6. Reinstall
pip uninstall $OLD_NAME -y
pip install -e ".[dev]"

echo "‚úÖ Renamed to $NEW_NAME!"
```
