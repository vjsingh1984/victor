# Task T5: Command Execution

## Objective
Execute and validate a series of shell commands with proper error handling.

## Requirements
Execute shell commands to:
1. Check system information (OS, Python version, memory)
2. Create temporary working directory
3. Create or download a sample text file
4. Perform text analysis (line/word/character counts)
5. Sort file contents and save to new file
6. Calculate checksums of both files
7. Compare files and report differences
8. Clean up temporary files

## Input
Step-by-step command execution instructions with validation requirements.

## Success Criteria
1. All commands executed successfully
2. Command output is captured and displayed
3. Output validation is performed for each step
4. Errors are handled gracefully
5. Temporary files are cleaned up
6. Checksums are correct

## Scoring
- 5 points: All commands executed with validation and error handling
- 4 points: All commands executed, minimal validation
- 3 points: Most commands executed, some failures
- 2 points: Basic commands only
- 1 point: Attempted but incomplete

## Notes
- LLM temperature: 0.3
- Timeout: 90 seconds
- Allowed tools: shell command execution
- Complexity: Simple
