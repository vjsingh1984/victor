# Task W1: Sequential Workflow

## Objective
Execute a sequential data processing workflow with conditional logic.

## Requirements
Execute a sequential data processing workflow:

### Step 1: Load data
Load data from `data.csv`

### Step 2: Validate data
Check for:
- Missing values
- Invalid data types
- Out-of-range values

### Step 3: Handle validation failures
If validation fails:
- Fill missing numeric values with mean
- Fill missing categorical values with mode
- Remove rows with invalid types

### Step 4: Transform data
- Normalize numeric fields to 0-1 range
- Encode categorical fields as one-hot
- Create derived features if applicable

### Step 5: Save results
Save processed data to `processed.csv`

## Constraints
- Each step should only execute if the previous step succeeded
- If any step fails, stop and provide an error message
- Log progress at each step

## Success Criteria
1. All steps execute in correct order
2. Conditional logic works correctly
3. Errors are handled gracefully
4. Output file is valid CSV
5. Data transformations are correct

## Scoring
- 5 points: Perfect execution with robust error handling
- 4 points: Correct execution with basic error handling
- 3 points: Workflow completes with some issues
- 2 points: Partial execution only
- 1 point: Attempted but incomplete

## Notes
- LLM temperature: 0.5
- Timeout: 180 seconds (3 minutes)
- Allowed tools: file_read, file_write, data analysis
- Complexity: Medium
