# Task T3: Web Research

## Objective
Conduct structured web research on a technical topic and produce a synthesis report.

## Requirements
Research the current state of WebAssembly (Wasm) adoption:
1. Search for recent articles on usage trends
2. Identify top languages that compile to WebAssembly
3. Find major companies using it in production
4. Summarize advantages and limitations
5. Compare performance vs JavaScript
6. List notable frameworks and toolchains

## Input
Research topic with specific information requirements and output format specification.

## Success Criteria
1. Relevant and recent information gathered
2. Multiple sources consulted
3. Findings are accurate and well-organized
4. Sources cited for claims
5. Executive summary is coherent
6. Conclusion provides forward-looking analysis

## Scoring
- 5 points: Comprehensive research with accurate citations and insights
- 4 points: Good coverage, minor gaps in sources or analysis
- 3 points: Adequate research, some areas superficial
- 2 points: Limited research, few sources
- 1 point: Minimal effort or inaccurate information

## Notes
- LLM temperature: 0.7
- Timeout: 120 seconds (2 minutes)
- Allowed tools: web search, web scraping
- Complexity: Medium
