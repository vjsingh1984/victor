# Task T1: File Operations

## Objective
Perform multi-step file operations using file system tools.

## Requirements
Perform the following file operations:

1. Create a directory structure: `output/{docs,images,data}`
2. Read the file at `README.md`
3. Extract all code blocks from the README
4. Write each code block to a separate file in `output/docs/`
5. Create a summary file listing all extracted files

## Input
The test will create a sample README.md file with multiple code blocks.

## Success Criteria
1. Directory structure created correctly
2. README.md is read successfully
3. All code blocks are extracted
4. Each code block is written to a separate file
5. Summary file lists all extracted files
6. No errors during execution

## Scoring
- 5 points: All operations completed correctly, robust error handling
- 4 points: All operations completed, minimal error handling
- 3 points: Most operations completed, some errors
- 2 points: Basic file operations only
- 1 point: Attempted but incomplete

## Notes
- LLM temperature: 0.3 (lower for precision)
- Timeout: 90 seconds
- Allowed tools: file_read, file_write, file_search
- Complexity: Simple
