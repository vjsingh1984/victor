# Task T4: Database Operations

## Objective
Perform a complete set of database operations including schema creation, data insertion, querying, and optimization.

## Requirements
Using SQLite:
1. Create a database with 4 related tables (users, orders, products, order_items)
2. Insert sample data (10 users, 20 orders, 15 products, 30 order items)
3. Execute analytical queries (top users, low stock, averages, filtered results)
4. Create indexes and analyze query plans
5. Export results to JSON

## Input
Detailed specification of tables, data requirements, queries, and expected output format.

## Success Criteria
1. All tables created with correct schemas and relationships
2. Sample data inserted without constraint violations
3. All queries return correct results
4. Index creation improves query performance
5. JSON export is valid and complete
6. No SQL errors during execution

## Scoring
- 5 points: All operations correct, efficient queries, good schema design
- 4 points: All operations completed, minor inefficiencies
- 3 points: Most operations completed, some query errors
- 2 points: Basic table creation and insertion only
- 1 point: Attempted but significant errors

## Notes
- LLM temperature: 0.3
- Timeout: 120 seconds (2 minutes)
- Allowed tools: database operations, file write
- Complexity: Medium
