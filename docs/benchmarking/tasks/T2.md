# Task T2: Git Workflow

## Objective
Execute a complete Git workflow including branching, merging, conflict resolution, and tagging.

## Requirements
Perform a full Git workflow:
1. Initialize repository with initial commit
2. Create and work on a feature branch
3. Create and apply a hotfix on a separate branch
4. Merge both branches to main (handling conflicts)
5. Create a tagged release

## Input
Step-by-step instructions for a 13-step Git workflow covering init, branching, committing, merging, and tagging.

## Success Criteria
1. Repository initialized correctly
2. Feature branch created and committed to
3. Hotfix branch created and committed to
4. Hotfix merged to main cleanly
5. Feature branch merged to main (conflicts resolved if any)
6. Release tag v1.0.0 created
7. Git log shows correct graph topology

## Scoring
- 5 points: All operations completed correctly, clean git history
- 4 points: All operations completed, minor issues in history
- 3 points: Most operations completed, merge issues
- 2 points: Basic git operations only
- 1 point: Attempted but incomplete

- LLM temperature: 0.3
