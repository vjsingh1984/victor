# Task A2: Performance Analysis

## Objective
Analyze Python code for performance bottlenecks and provide an optimized implementation.

## Requirements
Analyze a log processing function with multiple performance issues:
- O(n*m) nested iteration over log lines
- Quadratic duplicate removal
- Bubble sort implementation
- Inefficient regex usage
- Memory-heavy operations

## Input
Complete Python function source code with embedded performance anti-patterns.

## Success Criteria
1. Time complexity correctly analyzed for each section
2. Specific bottlenecks identified with Big-O notation
3. Memory usage issues identified
4. Optimized version provided with clear improvements
5. Performance improvement estimates are reasonable
6. Profiling approach recommended for validation

## Scoring
- 5 points: Complete analysis with correct Big-O, optimized code, and profiling guidance
- 4 points: Correct analysis, good optimized code, minor gaps
- 3 points: Major bottlenecks identified, partial optimization
- 2 points: Surface-level analysis, basic improvements
- 1 point: Incorrect analysis or trivial suggestions

## Notes
- LLM temperature: 0.5
- Timeout: 240 seconds (4 minutes)
- Complexity: Complex
- Tests ability to reason about algorithmic complexity and provide practical optimizations
