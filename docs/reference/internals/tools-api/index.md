# Victor Tools API Reference

Comprehensive API documentation for creating and registering tools in Victor.

---

## Quick Summary

This reference covers:
- **BaseTool Interface** - Abstract base class for all tools
- **Tool Registration** - How to register tools with the registry
- **Parameter Schema** - JSON Schema for tool parameters
- **Cost Tiers** - Tool execution cost classification
- **Access Modes** - Tool access control modes
- **Implementation Patterns** - Common tool implementation patterns
- **Tool Decorator** - Decorator-based tool creation
- **Advanced Features** - Dependencies, async execution, streaming
