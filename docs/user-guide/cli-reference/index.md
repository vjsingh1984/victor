# Victor CLI Reference

Complete command-line reference for Victor AI.

---

## Quick Summary

Victor AI provides a comprehensive CLI interface:

- **Core Commands** - Primary commands (chat, ask, serve)
- **Modes** - Agent modes (build, plan, explore)
- **API Keys** - Key management
- **Providers** - Provider management
- **Models** - Model operations
- **Tools** - Tool management
- **Configuration** - Config management
- **RAG Operations** - RAG-specific commands
- **Benchmarks** - Performance benchmarks
- **Workflows** - Workflow management
- **Sessions** - Session management
- **Verticals** - Vertical management
