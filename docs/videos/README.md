# Victor AI - Video Scripts

This directory contains comprehensive scripts for creating educational videos about Victor AI.

## Available Video Scripts

### 1. Introduction Video
**File:** `INTRODUCTION_VIDEO.md`
**Duration:** 10-12 minutes
**Target:** Developers, CTOs, Engineering Managers

**Topics:**
- What is Victor AI?
- Key problems it solves
- Main features and benefits
- Provider flexibility (21 providers)
- Quick start guide
- Call to action

**Use Case:** Marketing, onboarding, conference presentations

### 2. Advanced Features Video
**File:** `ADVANCED_FEATURES_VIDEO.md`
**Duration:** 15-18 minutes
**Target:** Experienced developers, DevOps engineers

**Topics:**
- Workflow system (YAML and Python)
- Custom tool development
- Multi-agent coordination
- Performance optimization
- Enterprise features (air-gapped, observability)
- Best practices

**Use Case:** Training, advanced user onboarding

### 3. Developer Guide Video
**File:** `DEVELOPER_GUIDE_VIDEO.md`
**Duration:** 12-15 minutes
**Target:** Plugin developers, contributors

**Topics:**
- Vertical architecture
- Creating custom verticals
- Adding tools and workflows
- Configuration system
- Testing and publishing
- Contributing guidelines

**Use Case:** Developer onboarding, hackathons, workshops

## Production Guidelines

### Pre-Production

**1. Script Review**
- [ ] Review script for accuracy
- [ ] Update with latest features
- [ ] Add current examples
- [ ] Verify command syntax

**2. Visual Planning**
- [ ] Create storyboard
- [ ] Design visual aids
- [ ] Prepare demo environments
- [ ] Test all demos

**3. Equipment Check**
- [ ] Quality microphone
- [ ] Screen recording software
- [ ] Video editing software
- [ ] Lighting setup

### Production

**1. Recording**
- Record in a quiet environment
- Use consistent audio levels
- Record screen in high resolution (1920x1080+)
- Use smooth cursor movements

**2. Voiceover**
- Professional voice talent recommended
- Clear, paced speech
- Consistent tone throughout
- Match timing to visuals

**3. Screen Recordings**
- Clean desktop (minimal distractions)
- Consistent terminal/editor settings
- Readable font sizes (14pt+)
- Highlight important elements

### Post-Production

**1. Editing**
- [ ] Trim unnecessary content
- [ ] Add transitions
- [ ] Insert visual aids
- [ ] Sync audio with visuals
- [ ] Add background music (subtle)

**2. Quality Check**
- [ ] Audio clarity
- [ ] Visual quality
- [ ] Timing and pacing
- [ ] Overall flow

**3. Export**
- Format: MP4
- Resolution: 1920x1080 (1080p)
- Codec: H.264
- Audio: AAC, 44.1kHz, 128kbps
- Frame rate: 30fps
