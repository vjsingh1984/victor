# HIGH-003: getattr() Audit and Categorization
# Total: 444 calls across codebase

## Category 1: Settings Access (Highest Priority)
### Pattern: getattr(self.settings, "field_name", default)
### Solution: Use VictorSettings typed fields directly
### Files:
- orchestrator_factory.py: ~50 calls
- orchestrator.py: ~35 calls  
- service_provider.py: ~15 calls
- slash_commands.py: ~10 calls

### Examples:
BEFORE: getattr(self.settings, "tool_cache_enabled", True)
AFTER:  self.settings.tool_cache_enabled

BEFORE: getattr(self.settings, "max_context_tokens", 100000)
AFTER:  self.settings.max_context_tokens

## Category 2: Tool Metadata Access
### Pattern: getattr(tool, "cost_tier", None)
### Solution: Use ToolMetadata.from_tool(tool)
### Files:
- metadata_registry.py: 12 calls
- ui/commands/tools.py: 4 calls
- tools/*.py: scattered

### Examples:
BEFORE: cost_tier = getattr(tool, "cost_tier", None)
AFTER:  metadata = ToolMetadata.from_tool(tool)
        cost_tier = metadata.cost_tier

## Category 3: Provider Response Access
### Pattern: getattr(response, "tokens_per_second", 0)
### Solution: Use ModelMetrics, UsageSummary models
### Files:
- slash_commands.py: 8 calls
- ui/commands/chat.py: 2 calls

### Examples:
BEFORE: tps = getattr(m, "tokens_per_second", 0)
AFTER:  metrics = ModelMetrics.from_dict(m)
        tps = metrics.tokens_per_second

## Category 4: Agent Component Access
### Pattern: getattr(self.agent, "component", None)
### Solution: Use AgentInfo.from_agent(agent)
### Files:
- slash_commands.py: 6 calls

### Examples:
BEFORE: mcp_client = getattr(self.agent, "mcp_client", None)
AFTER:  agent_info = AgentInfo.from_agent(self.agent)
        mcp_client = agent_info.mcp_client

## Category 5: Legitimate Dynamic Access (Keep as-is)
### Pattern: Reflection, plugin systems, AST inspection
### Files:
- tools/plugin_registry.py: getattr(mod, "__file__", None)
- tools/shared_ast_utils.py: getattr(node, "end_lineno", None)
- ui/tui/app.py: getattr(self._console, name) - delegation pattern
- tools/tool_names.py: getattr(ToolNames, attr) - enum introspection

### Justification: These are legitimate uses of reflection/introspection
