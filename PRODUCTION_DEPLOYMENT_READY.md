## ðŸš€ Deployment Instructions

### Quick Start (Pip)

```bash
# Clone repository
git clone https://github.com/victorai/victor.git
cd victor

# Install dependencies
pip install -e ".[all]"

# Run deployment
./scripts/deploy.sh production

# Verify deployment
./scripts/health_check.sh
```

### Docker Deployment

```bash
# Build image
cd scripts/docker
./build.sh 1.0.0 --push --scan

# Deploy
cd ../
./deploy.sh production
```

### Kubernetes Deployment

```bash
# Deploy to cluster
cd scripts/kubernetes
./deploy.sh production --namespace victor-prod
```

### Enable Features

```bash
# Enable agentic AI features

export VICTOR_ENABLE_SKILL_DISCOVERY=true
```
